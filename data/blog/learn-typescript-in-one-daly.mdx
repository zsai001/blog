## 基本的语法知识

### 类型注解
```typescript
let isDone: boolean = false;
let decLiteral: number = 6;
let name: string = "bob";
let list: number[] = [1, 2, 3];
```

### 接口
```typescript
interface Person {
  firstName: string;
  lastName: string;
}
let person: Person = {
    firstName: "John",
    lastName: "Doe"
};
```

### 类
```typescript
class Greeter {
  greeting: string;
  constructor(message: string) {
    this.greeting = message;
  }
  greet() {
    return "Hello, " + this.greeting;
  }
}
```

### 泛型
```typescript
function identity<T>(arg: T): T {
  return arg;
}
```

### 类型推断
```typescript
let x = 3;
```

### 类型断言
```typescript
let someValue: any = "this is a string";
let strLength: number = (someValue as string).length;
```

## 在项目中使用Typescript
- 使用`create-react-app`创建项目
- 使用`create-react-app my-app --template typescript`创建Typescript项目
- 使用`npx create-react-app my-app --template typescript`创建Typescript项目
